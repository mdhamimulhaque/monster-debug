What is the difference between Local storage and session storage?

  • Local storage data inside browser and when you will close your browser it doesn't remove data until clear local storage or change your browser. But Session storage data remove when you will close your open tab.
  • Local storage size 5-10 mb .one the other hand, session storage size 5 mb.

What is the difference between Global scope and block scope?

  • Global scope mean i can access its data from anywhere any time.example: if we will declare a variable with var keyword without any function, conditions or any kind of block it will be assessable. On the other hand, if we declare a variable inside any block and we don't have allow to use it's value outside this block it's called block scope. let and const maintain block scope.
